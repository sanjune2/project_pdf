# 🚇 AI 기반 유동인구 예측 데이터를 활용한 스마트 환기 운영 최적화 제안
> **서울교통공사 에너지 효율 개선 및 재정 건전성 확보를 위한 데이터 기반 차등형 공조 제어 시스템**

본 프로젝트는 서울 지하철 역사의 유동인구 변동성을 고려하지 않은 기존 일률적 냉방·환기(공조) 시스템의 한계를 극복하기 위해 기획되었습니다. 공공데이터를 기반으로 시간대별 유동인구 편차를 통계적으로 입증하고, 이에 따른 **'데이터 기반 차등형 스마트 스케줄링 제어 알고리즘'**을 제안합니다.

---

## 📌 1. 프로젝트 기획 스토리라인 (SCQA)

* **Situation (현황)** 서울교통공사는 지하철 내 쾌적한 환경 유지를 위해 대규모 냉방·환기 시스템을 상시 운영하고 있습니다.
* **Complication (문제점/위기)** 현재 시스템은 열차 무게, 공기질 센서, CCTV 데이터 등을 부분적으로 활용하지만, 유동인구가 급감하는 심야 시간 및 여유 시간대에도 일률적으로 풀가동되어 막대한 전력 과소비와 재정 적자를 심화시키고 있습니다.
* **Question (핵심 질문)** *“승객의 실질적인 쾌적성을 완벽히 보장하면서, 공실 운영으로 인한 전력 낭비를 어떻게 획기적으로 감축할 것인가?”*
* **Answer (해답)** **시간대별 유동인구 추이 예측 데이터**를 실시간 공조 시스템과 연동하여, 혼잡도에 따라 가동률을 자동으로 조절하는 **[차등형 스마트 스케줄링 시스템]**을 도입합니다.

## 📊 3. 데이터 분석 및 통계적 가설 검정

주피터 노트북(`스마트_환기_운영_최적화_제안.ipynb`) 내에서 공공데이터 포털의 최신 서울교통공사 승하차 인원 데이터를 활용하여 통계 분석을 수행했습니다.

### 💡 가설 설정
* **귀무가설 ($H_0$):** 시간대별(출퇴근 시간대 vs 비출퇴근 시간대) 평균 유동인구의 차이가 없다.
* **대립가설 ($H_1$):** 시간대별(출퇴근 시간대 vs 비출퇴근 시간대) 평균 유동인구는 통계적으로 유의미한 차이가 있다.

### 🧪 검정 방법
* **독립표본 t-검정 (Independent Samples t-test)**을 적용하여 출퇴근 피크 타임과 심야/여류 시간대의 유동인구 평균을 비교 분석했습니다.
* **결과 요약:** 유의수준 $\alpha = 0.05$ 기준 하에, 역별(강남역 등 유동인구 밀집 지역) 시간대별 편차가 통계적으로 매우 유의미함($p\text{-value} < 0.05$)을 확인하여 귀무가설을 기각했습니다. 이는 **차등형 공조 제어 시스템 도입의 명확한 데이터적 근거**가 됩니다.
